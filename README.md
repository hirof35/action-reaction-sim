# 作用反作用シミュレーション (Action & Reaction Simulation)

ニュートンの第3法則「作用・反作用の法則」を視覚的に体験・理解するための、Go言語（Ebitengine）製2D物理シミュレーションアプリです。

## 概要

このアプリは、質量の異なる2つのボールが衝突した際、お互いに「等しい大きさ」で「逆向き」の力を受け合う様子をリアルタイムに描写します。

- **物理エンジン不使用:** 運動量保存則に基づき、完全弾性衝突の数式をピュアなGo言語で実装しています。
- **力の可視化:** 衝突した瞬間に、お互いが受けた力（力積）のベクトルを黄色の矢印で一定時間描画します。質量が違っても受ける力の大きさ（矢印の長さ）が全く同じであることを直感的に確認できます。

## 動作環境

- Go 1.20 以上
- Ebitengine v2

## インストールと実行方法

1. **リポジトリのクローン、または作業用ディレクトリの作成**
```bash
   mkdir action-reaction
   cd action-reaction
Goモジュールの初期化と依存関係のインストールBash   go mod init action-reaction
   go get [github.com/hajimehoshi/ebiten/v2](https://github.com/hajimehoshi/ebiten/v2)
実行main.go を作成し、ソースコードを貼り付けたあと、以下のコマンドを実行します。Bash   go run main.go
操作方法[Space] キー: シミュレーションを初期状態（リセット）に戻し、ボールを再発射します。シミュレーションの観察ポイント質量の差と挙動:赤色（小）は質量 1.0、青色（大）は質量 3.0 に設定されています。衝突時、2つのボールに表示される矢印（力）の長さは同じですが、軽い赤色のボールの方が激しく吹き飛ばされます。これは $F=ma$（加速のしにくさは質量に比例する）の法則によるものです。コードのカスタマイズ:Reset() 関数内にある g.ball1.mass や g.ball2.mass の数値を書き換えることで、お互いの重さを同じにしたり、さらに極端な差をつけた時の挙動を実験できます。ライセンスMIT License
